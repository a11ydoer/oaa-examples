Accessibility Feature

  • A combo box enables the user to type in a field and at the same time chose a predefined value from a list. By using the keyboard the user can select an item from the list. After selection she will be able to type further characters in the field.

Keyboard Support

Key Function
Left/Right Arrow Move the carot left or right within the edit field
Alt+Up/Down Arrow Opens and closes the list.
Up and Down Arrow Move focus up and down the list. As focus moves the edit field is updated.
Page Up/Down Selects the next/previous pages item depending on the list size.
Escape Closes the dropdown list, returns focus to the edit field and does not change the current selection.
Enter Selects the current item on the list, updates the edit field, highlights the selectied item in the dropdown list, closes the dropdown list and returns focus to the input field.
Alphanumeric Characters Alphanumeric characters move focus to the next instance of a visivle node whose title begins with that printable letter.
