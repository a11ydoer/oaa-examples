Accessibility Features

  • The tree view container has a role=tree
  • Each node in a tree has the role=treeitem and should be a DOM child of tree.
  • A collection of treeitems to be expanded and collapsed are enclosed in a group.
  • Each tree node which can be expanded should have aria-expanded="false"
  • Each tree node which is expanded should have aria-expanded="true"

Keyboard Support

Tree View

Key Function
Up and down arrow Key move between visible nodes.
Left Arrow Key on an expanded node closes the node and on a closed or end node moves focus to the node's parent.
Right Arrow Key expands a closed node, moves to the first child of an open node, or does nothing on an end node.
Return Key performs the default action on end nodes.
Home Key moves to the top node in the tree view.
End Key moves to the top node in the tree view.
