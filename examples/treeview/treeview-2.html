Events

Event Description
keydown A keydown event listener is added to each element with role=treeitem to support moving focus between the treeitem with the keyboard.
click Click event listener is added to each element with role=treeitem to support opening and closing a leaf of the tree with the mouse.
focus Focus event listener is added to each element with role=treeitem to support visual styling of the treeitem when it recieves focus.
blur Blur event listener is added to each element with role=treeitem to support visual styling of the treeitem when it does not have focus.

Keyboard Support

Key Function
Arrow Key Focus and select the node. Any previous selections are cleared
Up arrow Key Move up to previous visible node.
Down arrow Key Move down to next visible node.
Left Arrow Key Closes the node on an expanded node, moves focus to the node's parent on a closed or end node.
Right Arrow Key Expands on a closed node, moves focus to the first child of an open node, or does nothing on an end node.
Return Key Performs the default action on end nodes.
Home Key Moves to the top node in the tree view.
End Key Moves to the last visible node in the tree view.

ARIA Roles

Role Usage
tree The tree view container has a role=tree.
treeitem Each node in a tree has the role=treeitem and should be a DOM child of tree.
group A collection of treeitems to be expanded and collapsed are enclosed in a element with role=group. The ul element has the default role of group.

ARIA Properties and States

Property/State Usage
aria-expanded
  • Each element with role=treeitem that can be expanded or contracted should have an aria-expanded attribute.
  • Each element with role=treeitem which is expanded should have aria-expanded="true".
  • Each element with role=treeitem which is collapsed should have aria-expanded="false".

tabindex values

Element Selector Values Purpose
[role=treeitem] 0 or -1 The treeitem with the current focus should have tabindex=0, all other treeitems should have a tabindex value of -1.
[role=treeitem] a -1 Remove any links that are descendants of a treeitem from the tab order of the page by changing tabindex value to -1 (e.g. the default value is 0).
